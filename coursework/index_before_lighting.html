To control this animation use the following:

Use "arrow up" to increase the satelite speed and "arrow down" to decrease the satelite speed, to set any speed change the variable "pwgl.sateliteSpeed" to the value you want to in the js terminal.

Use "arrow right" to increase the satelite orbit radius and "arrow left" to decrease the satelite orbit radius, to set any orbit radius change the variable "pwgl.orbitRadius" to the value you want to in the js terminal.

The satelite period is a result of the speed of the satelite and the radius of the orbit.

Use "w" to increase the satelite orbit inclination and "s" to decrease the satelite orbit inclination, to set any orbit radius change the variable "pwgl.orbitInclination" to the value you want to (in radians) or use the function "setInclination(angle)" with the angle (in degrees) in the js terminal.
